# Retail Sales Analysis using SQL

## Project Overview
This project analyzes retail sales data using SQL. The goal is to extract meaningful business insights, identify sales trends, and practice real-world SQL concepts .

## Dataset
The dataset contains information about:
1. Sales transactions
2. Customers
3. Products
4. Revenue generated from sales

## Project Files

* `schema.sql` – Database and table creation scripts
* `sample_data.sql` – Sample data insertion scripts
* `analysis_queries.sql` – SQL queries for business analysis
* `insights.md` – Key findings and recommendations

## SQL Concepts Used

* SELECT Statements
* WHERE, GROUP BY, HAVING
* ORDER BY
* Aggregate Functions
* Joins
* Subqueries
* Common Table Expressions (CTEs)
* Window Functions
* Ranking Functions

## Business Questions Answered

1. What are the top-selling products?
2. Which customers generate the highest revenue?
3. What are the monthly sales trends?
4. Which products contribute most to total sales?
5. What are the highest-value transactions?
6. How does revenue change over time?

## Tools Used
* PostgreSQL
* pgAdmin
* GitHub
