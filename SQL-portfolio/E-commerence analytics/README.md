## Project Overview

E-Commerce Analytics Platform is an advanced SQL project simulating a real-world online retail database. It covers the full analytics pipeline — from raw transactional data to actionable business insights — using pure SQL on PostgreSQL.

## Dataset Summary
1. 10 customers across 6 Indian cities (Mumbai, Delhi, Bangalore, Hyderabad, Chennai, Kolkata)
2. 12 products across 5 categories (Electronics, Clothing, Books, Home & Kitchen, Sports & Fitness)
3. 20 orders with realistic mixed statuses — Completed, Cancelled, Returned, Pending
4. 24 order line items with INR pricing
5. 10 product reviews with ratings from 1 to 5

## SQL Concepts Used
* WHERE, GROUP BY, HAVING
* ORDER BY
* Aggregate Functions
* Joins(INNER,LEFT,RIGHT OUTER)
* Subqueries
* Common Table Expressions (CTEs)
* Window Functions(LEAD(),LAG(),ROW_NUMBER())
